Cognitive Parallel

Inspired by Human Intelligence

How the human brain optimizes raw inputs to operate under a low metabolic energy limit.

Sensory Attention Selection

98% SILENT

Sensory neurons discard redundant stimuli at the periphery before firing signals to the cortex.

Sparse Sensory Activation

Imagine a student studying in a library. Their sensory organs receive constant physical waveforms: a humming ceiling fan, the ticking clock, ambient rustles.

Yet the brain registers **zero attention** to these redundant inputs. Metabolic energy is conserved until a novel event occurs—such as a door sliding open or a direct voice. At that moment, the brain fires a neural spike, wakes the cognitive core, and makes decisions.

Integration Layers

Technology Stack

Built on high-performance frameworks and models tailored for low-resource environments.

OpenCV

Manages high-frequency video capture, frame decodings, and delta comparisons.

YOLOv8 Nano

Ultralightweight object model. Loads into RAM and executes object checks rapidly on wake triggers.

FastAPI

Serves JSON metrics and Server-Sent Events (SSE) logs directly to dashboards with low network latency.

SQLite

Embedded document database saving event records, confidence ratios, and local frame links.

asyncio

Orchestrates multi-camera pipelines concurrently on a single thread event loop.

Edge Hardware

Optimized to run on CCTV nodes, Raspberry Pi configurations, or mobile phone microprocessors.

Functional Architecture

Core Modules

10 modular services structured to process spikes cleanly, saving power and computing latency.

1. Video Acquisition

Establishes real-time RTSP/RTMP endpoints with adaptive framerate handling.

2. Event Detection Engine

Low-cost background subtraction checker that spots pixel delta modifications.

3. Spike Event Queue

Buffers spike activations to handle packet bursts cleanly during peak motion.

4. Asynchronous Processor

Spins worker tasks concurrently to run detections without freezing video feeds.

5. Object Detection

Executes YOLOv8 checks on buffered frames to extract tags (human, vehicle, etc.).

6. Threat Analysis

Matches extracted tags against local alert rules and custom vector bounds.

7. Snapshot Generator

Caches the high-resolution frame matching active alert triggers as JPEG.

8. Alert System

Pushes live event snapshots to WebSockets, emails, and discord webhook targets.

9. Database Logging

Logs occurrences timestamps, labels, and inference confidence metrics in DB.

10. Analytics Dashboard

Visualizes cumulative power savings, camera logs, and alerts in the FastAPI UI.

Neuromorphic Principles

Emulating biological systems to restructure standard computer vision feeds.

Event-Driven Processing

Zero processor ticks spent when frames display static textures. Computations wake strictly when changes break limits.

Sparse Activation

Only a fraction of the neural pipeline resides in dynamic memory at once, replicating sparse brain signals.

Spiking Networks

Inputs represent temporal events (spikes) rather than continuous value arrays, minimizing processing footprint.

Asynchronous Streaming

Modules execute logic out-of-sync, resolving delays that freeze standard frame-based capture cycles.

Parallel Processing

Distributes independent video feed triggers across parallel CPU cores, ensuring scalable load management.

Low-Power Computing

Saves up to 92% CPU loads under idle periods, translating directly to massive cost benefits.
